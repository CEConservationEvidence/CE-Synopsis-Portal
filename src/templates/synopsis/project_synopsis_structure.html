How This Compiles

Chapter background appears once at the start of each chapter.
Intervention group (subheading) is a level-2 heading used to organize interventions.
Intervention background + synthesis appears before intervention evidence.
Assigned summaries compile into numbered study paragraphs and intervention references.
Key message supporting studies compile as numbered citations such as (1; 3).
Output order: Chapter -> Intervention group -> Intervention -> Key messages -> Study paragraphs -> References
