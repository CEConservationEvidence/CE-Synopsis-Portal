How the global reference database works
Shared library, deduplication, and how records reach a synopsis.
What this is

The global reference database is a shared master library for the whole portal. It stores canonical reference metadata that can be searched, edited, and linked into individual synopses.

How references get into the library
  • Upload a RIS, XML, or supported plain-text file directly into the library.
  • Import a RIS or plain-text file into a project. The system still creates or reuses a matching library record behind the scenes.
How duplicates are handled today

The system deduplicates library records using a hash based mainly on title, publication year, and DOI.

  • If the same reference is uploaded to the library again, it is skipped rather than imported twice.
  • If a project already has that reference, linking from the library reuses the existing project copy instead of creating another one.
What “Link to project” means

Linking does not move a reference out of the library. It creates a project-level working copy for that synopsis, grouped into a project batch so authors can screen and manage it locally.

What is shared vs what stays per synopsis
Shared in the global library
  • Core citation metadata
  • Searchable shared reference records
  • Import history at the library level
Managed per synopsis
  • Screening status
  • Reference folders
  • Screening notes and comments
  • Summary tabs, assignments, and evidence links
How this compares with the current EndNote workflow

It is similar to EndNote in that there is one central searchable pool of references that can be reused across multiple projects.

It is not yet a one-for-one EndNote replacement. Screening, synopsis folders, and summary work happen on project-specific copies after linking or import, and the global library component is still evolving.

When to use this page

Use the global library when you want to search shared references, tidy canonical metadata, or link existing references into a synopsis. If you just want to load a new search result set for one synopsis, you can also import directly into that project.
