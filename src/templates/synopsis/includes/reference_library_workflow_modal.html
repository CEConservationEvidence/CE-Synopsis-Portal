How the shared reference library works
What this page is for, how repeated references are handled, and how references reach a synopsis.
What this is

This is the shared library of references for the whole portal. It keeps one main record for each reference so that references can be searched, edited, and reused across different synopses.

How references get into the library
  • Upload a RIS, XML, or supported plain-text file directly into the library.
  • Import a RIS or plain-text file into a project. The system still creates or reuses a matching library record behind the scenes.
How repeated references are handled

The system tries to spot repeated references mainly by checking the title, year, and DOI.

  • If the same reference is uploaded to the library again, it is skipped rather than imported twice.
  • If a project already has that reference, linking from the library reuses the existing project copy instead of creating another one.
What “Link to project” means

Linking does not remove a reference from the shared library. It creates a copy for that synopsis so authors can screen it, place it in folders, add notes, and use it in summaries.

What is shared vs what stays per synopsis
Shared in the global library
  • Main citation details
  • Searchable shared reference records
  • Import history at the library level
Managed per synopsis
  • Screening status
  • Reference folders
  • Screening notes and comments
  • Summary tabs, assignments, and evidence links
How this compares with the current EndNote workflow

It is similar to EndNote in that there is one shared, searchable pool of references that can be reused across different projects.

It is not yet a full EndNote replacement. Screening, synopsis folders, and summary work happen inside each synopsis after linking or import, and this part of the portal is still developing.

When to use this page

Use the shared library when you want to search existing references, tidy the main reference details, or link references into a synopsis. If you only want to bring in a new search result set for one synopsis, you can also import directly into that project.
