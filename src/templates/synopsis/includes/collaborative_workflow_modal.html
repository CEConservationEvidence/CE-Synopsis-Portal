How collaborative editing works for this {{ document_label|lower }}

Use this guide for the live OnlyOffice session itself.

  1. Choose Open collaborative editor. If no shared session exists yet, the portal starts one. If a session is already live, the same button opens that current shared editor.
  2. Authors with edit access always use the same button from the portal, so everyone joins the same live {{ document_label|lower }} document.
  3. Use the editor normally while the session is open. The portal keeps that session attached to the current revision on this page.
  4. Using Back to {{ document_label|lower }} page only leaves that participant's browser tab. It does not close the shared session for anyone else.
  5. When all edits are complete, an author or manager can use Save current version and close shared editor from the {{ document_label|lower }} detail page. That requests a final save, creates a saved portal revision if needed, and closes the current shared session for everyone.
  6. Closing the shared session is not a permanent lock. Authors can open a fresh collaborative session later if more edits are needed.
  7. The advisory review deadline does not automatically close author editing. Collaborative editing stops when the feedback window is closed on the {{ document_label|lower }} detail page.
  8. After the session closes, return to the detail page to review the saved version, keep it in draft, or mark the current revision as final.
Collaborative editing is optional. If you prefer, you can still update the {{ document_label|lower }} by downloading the current revision, editing offline, and uploading a new revision manually.
