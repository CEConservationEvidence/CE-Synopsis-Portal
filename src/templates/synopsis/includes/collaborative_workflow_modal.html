How collaborative editing works for this {{ document_label|lower }}

Use this guide for the live OnlyOffice session itself.

  1. Choose Start collaborative edit to open one shared OnlyOffice session for the current {{ document_label|lower }} revision.
  2. Authors with edit access reopen the same live session from the portal while it is active, so everyone works in the same document.
  3. Use the editor normally while the session is open. The portal keeps that session attached to the current revision on this page.
  4. When one participant is finished, use Leave editor. That exits only that participant and does not close the shared session for anyone else.
  5. When all edits are complete, an author or manager can use Save and close session for everyone from the {{ document_label|lower }} detail page. That requests a final save and closes the shared session.
  6. After the session closes, return to the detail page to download the saved file, keep it in draft, or mark the current revision as final.
Collaborative editing is optional. If you prefer, you can still update the {{ document_label|lower }} by downloading the current revision, editing offline, and uploading a new revision manually.
