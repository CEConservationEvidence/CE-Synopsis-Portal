How {{ document_label|lower }} collaboration works

Use this workflow when authors need to work on one current {{ document_label|lower }} file together.

  1. Upload the latest {{ document_label|lower }} document on this page. It becomes the current working revision.
  2. Choose Start collaborative edit to open one shared OnlyOffice session for this {{ document_label|lower }}.
  3. Authors with edit access reopen the same live session from the portal while it is active, so everyone works in one place.
  4. When edits are complete, use Save and end session from the portal. That requests a final save and closes the shared session for everyone.
  5. The saved file remains the current revision here, where it can be downloaded, kept in draft, or marked final once agreed.
If the {{ document_label|lower }} is marked final, switch it back to draft before replacing it with a new revision. Closing the feedback window also closes collaborative editing for that document.
